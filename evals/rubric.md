# Blind judging rubric (/30)

Score each artifact on 6 criteria, 0–5 each. You are scoring framing+plan submissions for the SAME item; you do NOT know which method produced which — judge only on quality, do not guess methods.

1. **Framing depth** — grasps what the item fundamentally IS, beyond surface restatement.
2. **Scope & assumption handling** — assumptions surfaced and scoped, not silently baked in.
3. **Blind-spot value** — surfaces non-obvious risks / edge cases / better approaches a plain plan would miss.
4. **Prioritization** — ordered by what's load-bearing / what matters first.
5. **Signal-to-noise** — concise and decision-relevant, not padded ceremony.
6. **Decision handle** — gives a clear next action, or the single key question to resolve first.

**Calibration for crisp items:** a lean artifact that correctly avoids ceremony on a well-specified item should NOT be penalized on signal-to-noise; over-framing a simple item (a full matrix where one line would do) IS a signal-to-noise failure and should score low on #5.

**Output:** a markdown table — `label | c1 | c2 | c3 | c4 | c5 | c6 | total | one-line justification` — one row per artifact.
